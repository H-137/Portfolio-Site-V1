Welcome to Mattleopold.ml

My name is Matt Leopold! I am a high school senior intrested in computer science and game design.

Here you can find my current and past projects.

Cat-Jam

Cat-Jam is a side-scrolling Unity runner game with voxel graphics. My original concept was in a 2-D pixel style, but when I decided to transition to 3-D I wanted to continue to use the sprites I already made. This new format introduced me to voxel art, 3 dimensional pixel art. This was my first dive into Unity 3-D and the new challenges it brought.

Museum Breakin

Museum Break In is a 2-D turn based stealth game. I created this concept with two of my friends for a game design competition. The most challenging aspect of this game was detecting when the player is inside the view of the guards, alongside the related lighting system. This project was also created in unity. Check it out!

React Blackjack

Blackjack! This was my final project for computer science and my first time using the React framework. The greatest challenge was understanding how the javascript interacted with the html and css.

Pong

The classic pong game in a web format! This was my first project with unity. Complete with both a multiplayer and single player mode and pixelated explosions.

Html Bubble Wrap

A simple relaxing bubble wrap experience with calming music to go with it.

Surface Pro 5 Hackintosh

My first hackintosh project, running macos on a surface pro 5! It was definitely not an easy journey, but I learned how to use documentation to solve problems I face.
